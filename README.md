# [![Raider Robotics & Engineering Logo](images/logos/RR&E_logo.png)](https://www.wylieeastctsoboosterclub.com/robotics) Team 8858's Code Repository

This repository contains code for all active projects of [Team 8858 'Beast from the East'](https://www.thebluealliance.com/team/8858) including:

* Current [FRC](https://www.firstinspires.org/robotics/frc) Robot of Team 8858.
* [T-Shirt Cannon](Tshirt_Cannon/)
* general scripts that can be used for [analysis](analysis_scripts/) by parsing [The Blue Alliance's](https://www.thebluealliance.com) database
* [scouting tools](Scouting/)

FRC Code for prior years is stored in the [Archive Repository](https://github.com/RaiderRobotics-8858/FRC_Archive).
