# Vision Subsystem

Utilize limelight in order to read [AprilTags](https://docs.wpilib.org/en/stable/docs/software/vision-processing/apriltag/apriltag-intro.html) to turn the turret to score and to align the robot for climb in autonomous. Also used for pathplanner swerve movement.
